## Setup and Installation

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Android Studio
- Java Development Kit (JDK)

### Installation

1. Clone the repo
   ```bash
   git clone https://github.com/Abelion512/app-kantuk
   ```
2. Navigate to the project directory
   ```bash
   cd app-kantuk
   ```
3. Open the project in Android Studio and follow the prompts to sync Gradle.

## Usage

Build and run the application directly from Android Studio on an emulator or physical device.
